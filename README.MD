# Daily Calorie Tracker

**Author:** Navraj Amgai  
**Date:** 2025-11-13  

## Description
A simple Python CLI program that lets users enter meals and calories, calculates total and average calorie intake, compares it with a daily limit, and optionally saves the report to a text file.

## How to Run
1. Open a terminal in the project folder.
2. Run:
3. Follow the prompts to enter meal names, calories, and your daily limit.
4. View the summary and choose whether to save the report.
